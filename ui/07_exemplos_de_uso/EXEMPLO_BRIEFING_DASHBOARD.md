# Exemplo de Briefing para Agente

## Pedido

Crie uma página de dashboard executivo para uma solução de IA governada, com indicadores de uso, riscos, filas de atendimento e recomendações operacionais.

## Saída esperada do agente

- Cabeçalho institucional com título, contexto e status.
- Bloco de síntese executiva.
- KPIs com unidade e período.
- Gráfico de tendência para volume de interações.
- Tabela de riscos priorizados.
- Área de recomendações acionáveis.
- Estados de loading, erro e vazio.
- Layout responsivo.
- Português brasileiro com acentuação correta.

## Critérios de aceite

- A interface deve parecer Bravonix.
- A hierarquia visual deve ser clara.
- Os dados simulados devem indicar que são simulados.
- A visualização não pode esconder incertezas ou limitações.
