## 1. Normas ABNT Aplicáveis

### 1.1 Normas Fundamentais

| Norma | Título | Aplicação |
|-------|--------|-----------|
| **ABNT NBR 14724:2023** | Trabalhos acadêmicos — Apresentação | Estrutura geral, margens, paginação |
| **ABNT NBR 6024:2021** | Numeração progressiva das seções | Hierarquia de seções (1, 1.1, 1.1.1) |
| **ABNT NBR 6028:2021** | Resumos | Redação de sumários e ementas |
| **ABNT NBR 6027:2021** | Sumário | Formatação de índices |
| **ABNT NBR 6034:2021** | Índice | Referências cruzadas |
| **ABNT NBR 6023:2018** | Referências | Citações de legislação e jurisprudência |

### 1.2 Práticas Forenses dos Tribunais

#### STF (Supremo Tribunal Federal)
- Fonte: Times New Roman 12pt ou Arial 11pt
- Margens: Superior/Esquerda 3cm, Inferior/Direita 2cm
- Espaçamento: 1,5 entre linhas
- Citações longas: Recuo 4cm, fonte 10pt, espaçamento simples

#### STJ (Superior Tribunal de Justiça)
- Segue padrão ABNT NBR 14724
- Exige numeração progressiva até 4 níveis
- Parágrafos numerados em decisões

#### TJs (Tribunais de Justiça Estaduais)
- MAIORIA segue ABNT com variações locais
- Alguns exigem fonte específica (ex: TJSP prefere Arial)

## 2. Tipografia Jurídica

### 2.1 Fontes Aceitas

| Fonte | Uso Principal | Tamanhos | Observações |
|-------|---------------|----------|-------------|
| **Times New Roman** | Corpo de texto, petições | 12pt (corpo), 10pt (citações) | Padrão ouro jurídico |
| **Arial** | Documentos modernos, e-mails | 11pt (corpo), 10pt (citações) | Aceita pela maioria dos tribunais |
| **Calibri** | Documentos internos | 11pt (corpo) | Moderna, boa legibilidade |
| **Garamond** | Documentos formais | 12pt (corpo) | Elegante, tradicional |

### 2.2 Hierarquia Tipográfica

```
┌─────────────────────────────────────────────────────────────┐
│  TÍTULO PRINCIPAL (H1)                                      │
│  Times New Roman 16pt, Bold, Centralizado                   │
│  Espaçamento pós: 24pt                                      │
├─────────────────────────────────────────────────────────────┤
│  1. SEÇÃO DE PRIMEIRO NÍVEL (H2)                            │
│  Times New Roman 14pt, Bold, Alinhado à esquerda            │
│  Espaçamento pré: 24pt, pós: 12pt                           │

```

### 2.3 Espaçamento e Margens

#### Margens Padrão ABNT (A4)
```
┌──────────────────────────────────────────┐
│  ↑ 3cm (superior)                        │
│  ┌────────────────────────────────────┐  │
│  │                                    │  │
│  │   3cm (esquerda)                   │  │ ← 2cm (direita)
│  │                                    │  │
│  │   CORPO DO DOCUMENTO               │  │
│  │                                    │  │
│  │                                    │  │
│  └────────────────────────────────────┘  │
│  ↓ 2.5cm (inferior)                      │
└──────────────────────────────────────────┘
         21cm (largura A4)
```

| Elemento | Valor | Justificativa |
|----------|-------|---------------|
| Margem Superior | 3cm | Espaço para carimbo/anotações |
| Margem Esquerda | 3cm | Espaço para encadernação |
| Margem Direita | 2cm | Padrão ABNT |
| Margem Inferior | 2.5cm | Espaço para paginação |

#### Espaçamento entre Linhas
| Elemento | Espaçamento |
|----------|-------------|
| Corpo do texto | 1,5 |
| Citações longas (>3 linhas) | Simples (1,0) |
| Referências bibliográficas | Simples (1,0) |
| Notas de rodapé | Simples (1,0) |

#### Recuo de Parágrafo
- **Primeira linha**: 1,5cm (padrão forense brasileiro)
- **Parágrafo único/seção**: Sem recuo

### 2.4 Citações e Referências

#### Citação Curta (até 3 linhas)
Incorpora-se ao texto entre aspas duplas, seguida de referência entre parênteses.

> Exemplo: "A norma deve ser interpretada conforme a Constituição" (MENDES, 2023, p. 45).

#### Citação Longa (mais de 3 linhas)
```
┌─────────────────────────────────────────────────────────────┐
│  Recuo esquerdo: 4cm                                        │
│  Fonte: 10pt (menor que o corpo)                            │
│  Espaçamento: simples (1,0)                                 │
│  Sem aspas                                                  │
│  Bloco destacado                                            │
└─────────────────────────────────────────────────────────────┘
```

### 2.5 Notas de Rodapé

| Elemento | Configuração |
|----------|--------------|
| Fonte | Times New Roman 10pt |
| Espaçamento | Simples (1,0) |
| Separador | Linha horizontal 2cm, 1pt |
| Numeração | Contínua por documento |

## 3. Elementos Visuais e Estrutura

### 3.1 Numeração de Processos (CNJ)

Formato padrão: `NNNNNNN-DD.AAAA.J.TR.OOOO`

| Posição | Significado | Exemplo |
|---------|-------------|---------|
| NNNNNNN | Número sequencial | 0001234 |
| DD | Dígitos verificadores | 56 |
| AAAA | Ano de ajuizamento | 2025 |
| J | Órgão (Justiça) | 8 = TJSP |
| TR | Tribunal | 02 = 2ª Câmara |
| OOOO | Origem | 0001 = 1ª Vara |

**Exemplo**: `0001234-56.2025.8.26.0001`

### 3.2 Cabeçalhos de Petições

```
┌─────────────────────────────────────────────────────────────┐
│                     [LOGO DO ÓRGÃO]                         │
│                                                             │
│              NOME DO ÓRGÃO/JUIZO/TURMA                      │
│              Processo nº NNNNNNN-DD.AAAA.J.TR.OOOO          │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│              [NOME DA PEÇA]                                 │
│              (ex: PETIÇÃO INICIAL, CONTESTAÇÃO, etc.)       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 Estrutura de Peças Jurídicas

#### Petição Inicial (Padrão CPC/2015 - Art. 319)
```
1. PREÂMBULO
   └── Endereçamento ao Juízo
   
2. QUALIFICAÇÃO DAS PARTES
   ├── Autor (nome, estado civil, profissão, RG, CPF, endereço)
   └── Réu (mesmos dados)
   
3. DOS FATOS
   └── Narrativa cronológica e fundamentada
   
4. DO DIREITO
   ├── Fundamentação legal
   ├── Jurisprudência
   └── Doutrina
   
5. DOS PEDIDOS
   ├── Pedido principal
   ├── Pedidos alternativos
   ├── Requerimentos (provas, intimações)
   └── Valor da causa
   
6. DATAS E ASSINATURAS
   ├── Local e data
   └── Assinatura do advogado (OAB)
```

#### ETP (Estudo Técnico Preliminar) - Lei 14.133/2021
```
1. Descrição da necessidade
2. Área requisitante
3. Soluções de mercado
4. Contratações anteriores
5. Pesquisa de preços
6. Estimativa de quantidades
7. Modelo de gestão
8. Critérios de seleção
9. Riscos
10. Sustentabilidade
```

```python
# backend/apps/intelligent_assistant/services/pdf_service.py

class JuridicalPDFService(PDFService):
    """Extensão para documentos jurídicos com conformidade ABNT."""
    
    JURIDICAL_CSS = """
    @page {
        size: A4;
        margin: 3cm 2cm 2.5cm 3cm;
        @top-center {
            content: element(header);
            font-size: 9pt;
            color: #666;
        }
        @bottom-center {
            content: "Página " counter(page) " de " counter(pages);
            font-size: 9pt;
            color: #666;
        }
    }
    
    /* Numeração progressiva ABNT NBR 6024 */
    h1 { counter-reset: h2; }
    h2 { counter-reset: h3; }
    h3 { counter-reset: h4; }
    
    h1::before {
        counter-increment: h1;
        content: counter(h1) ". ";
    }
    
    h2::before {
        counter-increment: h2;
        content: counter(h1) "." counter(h2) ". ";
    }
    
    h3::before {
        counter-increment: h3;
        content: counter(h1) "." counter(h2) "." counter(h3) ". ";
    }
    """
```

## 6. Checklist de Conformidade

### 6.1 Checklist Tipográfico

- [ ] Fonte: Times New Roman 12pt ou Arial 11pt
- [ ] Títulos: Hierarquia correta (H1: 16pt, H2: 14pt, H3: 12pt)
- [ ] Espaçamento entre linhas: 1,5 (corpo), 1,0 (citações longas)
- [ ] Recuo de parágrafo: 1,5cm na primeira linha
- [ ] Margens: Superior/Esquerda 3cm, Inferior/Direita 2cm
- [ ] Citações longas: Recuo 4cm, fonte 10pt

### 6.2 Checklist Estrutural

- [ ] Numeração progressiva conforme ABNT NBR 6024
- [ ] Sumário com links para seções
- [ ] Paginação (rodapé, centralizada ou direita)
- [ ] Cabeçalho com identificação do documento
- [ ] Assinatura com dados completos (nome, OAB, contato)

### 6.3 Checklist Jurídico

- [ ] Numeração do processo no formato CNJ
- [ ] Qualificação completa das partes
- [ ] Fundamentação legal atualizada
- [ ] Jurisprudência citada com referência completa
- [ ] Pedidos claros e fundamentados
- [ ] Valor da causa indicado (quando aplicável)

### 6.4 Checklist PDF/Impressão

- [ ] PDF gerado com fontes incorporadas
- [ ] Links funcionais no PDF (sumário, referências)
- [ ] Impressão em A4 preservando margens
- [ ] Elementos de UI escondidos na impressão
- [ ] Assinaturas em página separada (evitar quebra)

```python
# Backend: Validação de blueprint jurídico
def validate_juridical_blueprint(blueprint: DocumentBlueprint) -> list:
    """Valida se blueprint segue padrões jurídicos."""
    errors = []
    
    # Tipografia
    if blueprint.pdf_font_family not in ['Times New Roman', 'Arial', 'Calibri']:
        errors.append(f"Fonte '{blueprint.pdf_font_family}' não recomendada para documentos jurídicos")
    
    if blueprint.pdf_font_size not in ['11pt', '12pt']:
        errors.append(f"Tamanho '{blueprint.pdf_font_size}' fora do padrão (use 11pt ou 12pt)")
    
    if blueprint.pdf_line_height not in ['1.5', '1.15', '1.3']:
        errors.append(f"Espaçamento '{blueprint.pdf_line_height}' fora do padrão ABNT")
    
    # Margens
    if blueprint.pdf_page_margin_top != '3cm':
        errors.append("Margem superior deve ser 3cm (padrão ABNT)")
    
    if blueprint.pdf_page_margin_left != '3cm':
        errors.append("Margem esquerda deve ser 3cm (padrão ABNT)")
    
    # Recuo de parágrafo
    if blueprint.pdf_paragraph_indent not in ['1.25cm', '1.5cm', '2cm']:
        errors.append("Recuo de parágrafo deve ser 1.25cm, 1.5cm ou 2cm")
    
    return errors
```
